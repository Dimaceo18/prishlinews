# Minsk News Telegram Bot

Простой бот предложки для новостного канала.

Функции:
- 4 кнопки меню
- прием текста, фото, видео
- пересылка сообщений в группу редакции

## Переменные окружения (Render)

BOT_TOKEN=токен от BotFather
ADMIN_CHAT_ID=id группы редакции

## Запуск

Render автоматически выполнит:

pip install -r requirements.txt
python bot.py
